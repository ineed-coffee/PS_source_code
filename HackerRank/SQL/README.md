# SQL Grammar

---

### `Modulation`

```mysql
SELECT *
FROM TABLE_NAME
WHERE FIELD%NUM=REMAIN;
```

​		

### `Unique value` 

```mysql
SELECT DISTINCT(FIELD_NAME)
FROM TABLE_NAME;
```

​	

### `Counting` 

```mysql
SELECT COUNT(FIELD_NAME)
FROM TABLE_NAME;
```

​	

### `Length of string` 

```mysql
SELECT LENGTH(FIELD_NAME)
FROM TABLE_NAME
```

​	

### `Sort`

```mysql
SELECT FEILD1
FROM TABLE_NAME
ORDER BY FIELD2 ASC , FIELD3 DESC....
```

​	

### `Limiting rows` 

```mysql
SELECT FIELD_NAME
FROM TABLE_NAME
LIMIT ROW_CNT
```
